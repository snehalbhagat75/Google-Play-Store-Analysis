

# 📱 Google Play Store Analysis

## 📌 Project Overview

This project explores the Google Play Store dataset to analyze app performance, user ratings, installs, reviews, pricing, and category-wise trends. Using Python for data cleaning, exploratory data analysis (EDA), and visualization, the project uncovers meaningful insights that can help understand user preferences and app market dynamics.

---

## 🎯 Objectives

- Clean and preprocess Google Play Store data
- Analyze app categories and distribution
- Explore user ratings and reviews
- Identify the most installed applications
- Compare free vs paid apps
- Analyze pricing trends
- Discover relationships between installs, ratings, and reviews
- Generate business insights through visualizations

## 💡 Key Insights

- Most apps on the Play Store are free.
- A few categories dominate the total number of apps.
- Highly installed apps generally receive a large number of reviews.
- Most applications have ratings between 4.0 and 4.5.
- Paid apps represent a smaller share of the marketplace compared to free apps.
